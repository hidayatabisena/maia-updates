## Panduan Setup

Photoshoot AI Tools
1. Akses menu admin
2. Navigasikan ke AI Tools > Product Photoshoot
3. Masukkan API Key Fal AI jika belum terhubung.
4. Klik Save untuk menyimpan pengaturan.
